<h1>About the Project -</h1>
EduQuiz Hub is a educational platform developed with HTML, CSS, JavaScript, PHP, and MySQL. Users can log in to access diverse courses, including HTML, CSS, JavaScript, React, Node.js, PHP, Android, and more. They can engage with interactive quizzes for each course, featuring 5 questions per quiz, and receive scores to assess their learning. Additional platform features include the ability for users to edit their profiles, provide ratings for the platform, view their quiz scores to track progress, and sign out when done.
